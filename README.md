# God's Own Trails 🌴

A beautiful, responsive travel blog showcasing the most stunning destinations in Kerala, India's "God's Own Country."

## Overview

God's Own Trails is a modern web application that explores and celebrates the natural beauty, cultural heritage, and adventure opportunities across Kerala's diverse landscapes. From backwaters to mountains, beaches to waterfalls - discover Kerala's most enchanting travel destinations.

## 🗺️ Featured Destinations

- **Alleppey** - The Venice of the East with serene backwaters
- **Athirappilly** - Majestic waterfalls surrounded by lush greenery
- **Bekal** - Historic fort overlooking the Arabian Sea
- **Gavi** - Hidden gem among Western Ghats mountains
- **Kovalam** - Picturesque crescent beach with lighthouse
- **Kumarakom** - Backwater paradise and bird sanctuary
- **Munnar** - Misty tea plantation hills
- **Poovar** - Island lagoon with golden sand
- **Thekkady** - Wildlife sanctuary and spice plantations
- **Vagamon** - Cool hill station with scenic views
- **Varkala** - Dramatic cliffs and spiritual atmosphere
- **Wayanad** - Adventure hub with waterfalls and wildlife

## 🎨 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Built with Bootstrap 5 for clean, professional styling
- **Fast & Lightweight** - Pure HTML, CSS, and JavaScript
- **Smooth Navigation** - Easy-to-use navbar with smooth scrolling

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/travel-blog-main.git
cd travel-blog-main
```

2. Open the project in your browser:
   - Simply open `index.html` in your favorite web browser
   - Or use a local server like Live Server (VS Code extension)

### Using Live Server (Recommended)

1. Install the Live Server extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
